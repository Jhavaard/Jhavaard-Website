DesktopApp Project

Year 2017, College Project

DesktopApp Welcome Form

The Welcome form is the first page the user will see when running the DesktopApp. The User has a choice to select from Student or Lecture depending who the user might be.

DesktopApp Lecture Login Form

If the user selects Lecture, then this is form the user will see next. This is the Lecture login form. The user can either enter his/her credentials or create an account to login.

DesktopApp Lecture Main Form

After the Lecture successfully logs into the system, the user will be taken to the Lecture Main form. Here the user being the lecture in this case, can create, edit and delete a test. The lecture can also view all test, see his/her profile, look at student test results or simply logout of the system. Lectures can create many tests for a student for as they see fit.

DesktopApp Welcome Form

If the user selects Student, then this is form the user will see next. This is the Student login form. The user can either enter his/her credentials or create an account to login.

DesktopApp Lecture Login Form

After the Student successfully logs into the system, the Student will be taken to the Student Main form. Here the Student can take a test. The Student can also see his/her profile, look at student test results or simply logout of the system.

DesktopApp Lecture Main Form

In this form the Student can choose the test that he/she wents to undertake. These are created by different lecture with varying subjects.

DesktopApp Welcome Form

Once the Student selects a test to undertake, this form appears. This is the test form. All questions are MCQ. Once a question is ticked a stroke is placed on that question to remind the Student that he/she as already answered that question. MCQ is displaced in a scroll view format.

DesktopApp Lecture Login Form

After the Student successfully completes the test, the student is brought to the Feedback form. Here the student can see what they got right or wrong. A green tick is shown for the correct answer and a red cross for the wrong. Students percentage and result is also shown.

DesktopApp Lecture Login Form

This is the Result form. Here the student can see all of his/her pass test results. The test name, mark, overall class score, the lecture who created the test and whether the pass or failes is listed.
